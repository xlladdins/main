XLL.RANGE.TAKE function

Return a new range or modify the range if a handle is given.

Description

If the absolute value of n is greater than or equal to the number of rows of the range the operation returns the entire range. If n is 0 the result is xltypNil: the Excel notion of an empty range.

Syntax

XLL.RANGE.TAKE(n, range)

  • n is the number of elements to take from the front rows (n positive) or back rows (n negative) of a range.

  • range is a range or a handle to a range.
