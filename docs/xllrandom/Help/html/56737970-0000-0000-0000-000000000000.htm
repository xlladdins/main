RANDOM.DISTRIBUTION.BERNOULLI function

Create a handle to a random number generator called by RANDOM.BOOL

Description

Generate TRUE with probability p and FALSE with probability 1 - p.

Syntax

RANDOM.DISTRIBUTION.BERNOULLI(engine, p)

  • engine specifies what engine to use from the RANDOM_ENGINE_* enumeration.

  • p is the probability of returning a TRUE variate.
